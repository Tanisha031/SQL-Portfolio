# 🗄️ SQL-Portfolio 

## 📌 Repository Description

This repository contains a structured collection of MySQL practice scripts, mini projects, advanced SQL queries, and database design case studies. It is designed to build strong skills in database management, data analysis, and real-world business problem-solving using SQL.

The content ranges from basic SQL concepts to advanced topics such as joins, subqueries, window functions, indexing, views, and data cleaning techniques. It also includes multiple mini and advanced projects based on real-world business scenarios.

---

## 📂 1. MYSQL Workbooks

This folder contains structured SQL scripts covering core to advanced MySQL concepts.

**Topics include:**
- Data Types and Operators  
- SELECT, INSERT, UPDATE, DELETE commands  
- WHERE, GROUP BY, HAVING, ORDER BY, LIMIT  
- DISTINCT, Aliases, Logical & Comparison Operators  
- Joins (INNER, LEFT, RIGHT, CROSS, SELF, OUTER)  
- Views (CREATE, DROP, UPDATE)  
- Indexing (CREATE, DROP INDEX)  
- Table operations and constraints  
- Set operations  

👉 **Purpose:** Build strong foundation and advanced understanding of SQL queries and database operations.
